# Vimium Cheat Sheet

## Navigation
| Key | Action |
|-----|--------|
| `j` / `k` | Scroll down / up |
| `h` / `l` | Scroll left / right |
| `gg` | Go to top of page |
| `G` | Go to bottom of page |
| `d` / `u` | Scroll half page down / up |
| `zH` / `zL` | Scroll all the way left / right |

## Link Hints
| Key | Action |
|-----|--------|
| `f` | Open link in current tab |
| `F` | Open link in new tab |
| `yf` | Copy link URL to clipboard |

## Tabs
| Key | Action |
|-----|--------|
| `J` / `gT` | Go one tab left |
| `K` / `gt` | Go one tab right |
| `g0` / `g$` | Go to first / last tab |
| `t` | Create new tab |
| `x` | Close current tab |
| `X` | Restore closed tab |
| `T` | Search through open tabs |
| `^` | Go to previously visited tab |
| `yt` | Duplicate current tab |
| `<<` / `>>` | Move tab left / right |

## History
| Key | Action |
|-----|--------|
| `H` | Go back in history |
| `L` | Go forward in history |

## Search / Find
| Key | Action |
|-----|--------|
| `/` | Enter find mode |
| `n` / `N` | Next / previous match |

## Omnibar (Vomnibar)
| Key | Action |
|-----|--------|
| `o` | Open URL / search |
| `O` | Open URL / search in new tab |
| `b` | Open bookmark |
| `B` | Open bookmark in new tab |

## Marks
| Key | Action |
|-----|--------|
| `ma` | Set local mark `a` |
| `` `a `` | Jump to local mark `a` |
| `mA` | Set global mark `A` |
| `` `A `` | Jump to global mark `A` |
| `` ` `` `` ` `` | Jump back to position before last jump |

## Misc
| Key | Action |
|-----|--------|
| `yy` | Copy current URL |
| `p` / `P` | Open clipboard URL in current / new tab |
| `i` | Enter insert mode (ignore Vimium) |
| `v` | Enter visual mode |
| `V` | Enter visual line mode |
| `gi` | Focus first text input on page |
| `r` | Reload page |
| `gs` | View page source |
| `?` | Show help / all keybindings |

## Visual Mode (after `v`)
| Key | Action |
|-----|--------|
| `h/j/k/l` | Extend selection |
| `w` / `b` | Forward / backward by word |
| `y` | Yank (copy) selection |
| `Esc` | Exit visual mode |

> **Tip:** Press `Esc` to cancel any partial command or exit any mode. Press `?` anytime to see the full built-in help.
