# macOS Shortcuts Cheat Sheet

> **Note:** These show the standard macOS keys (⌘ = Cmd). If you have modifier keys swapped, see your machine config in `machines/`.

## Window Management
| Key | Action |
|-----|--------|
| `Cmd+m` | Minimize window |
| `Cmd+h` | Hide app |
| `Cmd+Option+h` | Hide all other apps |
| `Cmd+q` | Quit app |
| `Cmd+w` | Close window/tab |
| `Cmd+Ctrl+f` | Toggle fullscreen |
| `Cmd+Tab` | Switch apps |
| `Cmd+`` ` `` | Switch windows of same app |

## Spaces & Mission Control
| Key | Action |
|-----|--------|
| `Ctrl+↑` | Mission Control |
| `Ctrl+↓` | App windows |
| `Ctrl+←` / `→` | Switch space/desktop |
| `F11` | Show desktop |

## Screenshots
| Key | Action |
|-----|--------|
| `Cmd+Shift+3` | Screenshot full screen |
| `Cmd+Shift+4` | Screenshot selection |
| `Cmd+Shift+4+Space` | Screenshot window |
| `Cmd+Shift+5` | Screenshot/record toolbar |

## Finder
| Key | Action |
|-----|--------|
| `Cmd+Shift+g` | Go to folder |
| `Cmd+Shift+.` | Toggle hidden files |
| `Cmd+Delete` | Move to trash |
| `Cmd+Shift+Delete` | Empty trash |
| `Cmd+i` | Get info |
| `Cmd+d` | Duplicate |
| `Space` | Quick Look |
| `Cmd+Option+v` | Move (cut + paste) |

## Text Editing
| Key | Action |
|-----|--------|
| `Cmd+c` / `v` / `x` | Copy / paste / cut |
| `Cmd+z` / `Shift+z` | Undo / redo |
| `Cmd+a` | Select all |
| `Option+Delete` | Delete word backward |
| `Cmd+Delete` | Delete to beginning of line |
| `Option+←` / `→` | Move by word |
| `Cmd+←` / `→` | Move to line start / end |
| `Cmd+↑` / `↓` | Move to document start / end |
| Hold `Shift` + above | Select while moving |

## Spotlight / Search
| Key | Action |
|-----|--------|
| `Cmd+Space` | Spotlight search |
| `Cmd+f` | Find in app |

## System
| Key | Action |
|-----|--------|
| `Cmd+,` | App preferences |
| `Cmd+Option+Esc` | Force quit dialog |
| `Ctrl+Cmd+q` | Lock screen |
| `Cmd+Shift+q` | Log out |
| `Ctrl+Power` | Restart/sleep/shutdown dialog |
