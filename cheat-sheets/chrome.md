# Chrome Cheat Sheet (macOS)

## Tabs
| Key | Action |
|-----|--------|
| `Cmd+t` | New tab |
| `Cmd+w` | Close tab |
| `Cmd+Shift+t` | Reopen closed tab |
| `Ctrl+Tab` | Next tab |
| `Ctrl+Shift+Tab` | Previous tab |
| `Cmd+1-8` | Go to tab by number |
| `Cmd+9` | Go to last tab |
| `Cmd+Option+→` / `←` | Next / previous tab |

## Windows
| Key | Action |
|-----|--------|
| `Cmd+n` | New window |
| `Cmd+Shift+n` | New incognito window |
| `Cmd+Shift+m` | Switch profile / account |
| `Cmd+q` | Quit Chrome |
| `Cmd+m` | Minimize window |
| `Cmd+h` | Hide Chrome |

## Navigation
| Key | Action |
|-----|--------|
| `Cmd+l` | Focus address bar |
| `Cmd+[` / `]` | Back / forward |
| `Cmd+r` | Reload |
| `Cmd+Shift+r` | Hard reload (bypass cache) |
| `Cmd+↑` / `↓` | Scroll to top / bottom |
| `Space` / `Shift+Space` | Scroll down / up |
| `Cmd+Option+i` | Open DevTools |
| `Cmd+Option+j` | Open DevTools console |
| `Cmd+Option+u` | View source |

## Find
| Key | Action |
|-----|--------|
| `Cmd+f` | Find on page |
| `Cmd+g` | Next match |
| `Cmd+Shift+g` | Previous match |
| `Esc` | Close find bar |

## Bookmarks
| Key | Action |
|-----|--------|
| `Cmd+d` | Bookmark current page |
| `Cmd+Shift+d` | Bookmark all open tabs |
| `Cmd+Option+b` | Open bookmark manager |
| `Cmd+Shift+b` | Toggle bookmarks bar |

## Zoom
| Key | Action |
|-----|--------|
| `Cmd+=` | Zoom in |
| `Cmd+-` | Zoom out |
| `Cmd+0` | Reset zoom |

## Address Bar
| Key | Action |
|-----|--------|
| `Cmd+l` | Select URL |
| `Cmd+Enter` | Open in new tab |
| `Shift+Enter` | Open in new window |
| type then `Tab` | Search within site |

## Misc
| Key | Action |
|-----|--------|
| `Cmd+Shift+Delete` | Clear browsing data |
| `Cmd+,` | Open settings |
| `Cmd+y` | Open history |
| `Cmd+j` | Open downloads |
| `Cmd+Shift+j` | Open downloads (alt) |
| `Cmd+p` | Print page |
| `Cmd+Option+f` | Google search (if extension) |
| `Cmd+Ctrl+f` | Toggle fullscreen |
| `F5` / `Cmd+r` | Reload |
| `Cmd+s` | Save page as |
