# Mac Mini Setup

## Hardware
- **Computer:** Mac Mini
- **Keyboard:** PC (Windows) keyboard

## Modifier Key Remapping
Swapped via **System Settings > Keyboard > Keyboard Shortcuts > Modifier Keys**:

| Physical Key | Mapped To |
|-------------|-----------|
| Ctrl (PC) | Cmd (⌘) |
| Win/Super (PC) | Ctrl (⌃) |

This means all "Cmd" shortcuts are hit with the **physical Ctrl key** (bottom-left), matching standard PC muscle memory.

## Effective Shortcuts (what you physically press)

Since Cmd and Ctrl are swapped, the physical keys differ from what macOS docs say:

| Physical Keys | macOS Sees | Action |
|--------------|------------|--------|
| `Ctrl+c` | `Cmd+c` | Copy |
| `Ctrl+v` | `Cmd+v` | Paste |
| `Ctrl+x` | `Cmd+x` | Cut |
| `Ctrl+z` | `Cmd+z` | Undo |
| `Ctrl+Shift+z` | `Cmd+Shift+z` | Redo |
| `Ctrl+a` | `Cmd+a` | Select all |
| `Ctrl+s` | `Cmd+s` | Save |
| `Ctrl+q` | `Cmd+q` | Quit app |
| `Ctrl+w` | `Cmd+w` | Close window/tab |
| `Ctrl+t` | `Cmd+t` | New tab |
| `Ctrl+Space` | `Cmd+Space` | Spotlight |
| `Ctrl+Tab` | `Cmd+Tab` | App switcher |
| `Win+c` | `Ctrl+c` | Terminal interrupt (SIGINT) |
